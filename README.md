# 🎥 Videon - YouTube Video Downloader & Trimmer

**Videon** is a powerful, modern desktop application that allows you to download and trim YouTube videos quickly and easily. The app supports cross-platform environments (Windows, macOS, Linux) and comes bundled with the necessary tools to run out-of-the-box without complex setup.

---

## ✨ Key Features

- **High-Quality Downloads:** Support downloading videos in various resolutions (from 360p, 720p, 1080p Full HD up to 4K).
- **Smart Trimming:** Crop videos directly before downloading by choosing a custom time range or auto-trimming by video **Chapters**.
- **Audio Extraction:** Support extracting and downloading high-quality audio in `.mp3` or `.m4a` format.
- **Standalone Package (All-in-one):** Bundles optimized versions of `yt-dlp` and `ffmpeg` for each platform. Users can run the application immediately after downloading.
- **Download Manager:** Monitor download status, speed, and manage your downloaded files with an intuitive dashboard.
- **Modern User Interface (UI):** Minimalist UI designed for optimal user experience, utilizing the Inter font for perfect Vietnamese and English rendering.
- **Auto-Update System:** Automatically checks and notifies you when a new version is available.

## 🚀 Installation Guide

Visit the **[Releases](https://github.com/yorupngg-creator/Videon/releases)** section of this repository to download the latest version for your operating system:

### 1. Windows (`.exe`)
- Download the setup executable.
- Double-click the installer to install automatically. The app will launch once the installation completes.

### 2. macOS (`.dmg`)
- Download the `.dmg` file.
- Double-click the `.dmg` file and drag the **Videon** icon into your **Applications** folder.
- **First-time Opening Note:** 
  Since the app is unsigned, macOS will display a security warning. Follow these steps to bypass it:
  1. Right-click (or hold `Control` + click) the app in your *Applications* folder and select **Open**.
  2. Click **Open** again in the confirmation dialog.
  *(Alternatively, go to **System Settings > Privacy & Security**, scroll to the bottom, and click **Open Anyway**).*

### 3. Linux (`.tar.gz`)
- Download the `.tar.gz` archive.
- Extract the archive and run the executable `videon` in the extracted directory.
